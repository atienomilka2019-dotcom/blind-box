# 盲盒福袋搭配工具 — 设计规格书

## 概述

合并现有 `blind-box.html` 和 `blindbox.html` 的优点，重新设计一个功能更强的盲盒福袋搭配网页工具。核心能力：用户填入产品信息、等级比例、利润率后，系统自动计算每个产品的整数权重和单抽价格；同时给出各等级保底参考。

## 技术方案

- 单文件 HTML（HTML + CSS + Vanilla JS），零外部依赖
- 数据持久化：`localStorage`
- Excel 导出使用纯 JS 生成（无第三方库，用 CSV UTF-8 BOM 格式，Excel 可直接打开）
- 暗色/亮色主题可切换
- 响应式布局

### 4. 核心计算算法

**输入：** 产品列表（含成本、等级）、等级比例、总权重 W、目标利润率 M

**步骤：**

```
1. 按等级比例分配等级总权重：
   sum = ratio_rare + ratio_europe + ratio_normal
   tierWeight[稀有] = W * ratio_rare / sum
   tierWeight[欧皇] = W * ratio_europe / sum
   tierWeight[普通] = W * ratio_normal / sum

2. 每个等级内部，按成本反比分配权重：
   — 高成本产品获得较低权重（控制亏损风险）
   — 各产品原始权重 = 等级总权重 × (1/成本) / Σ(1/成本)

3. 整数化（floor + 余数分配）：
   — 所有产品权重取 floor
   — 余数从大到小排序
   — 剩余权重按余数大小分配给排名靠前的产品
   — 最终所有权重为正整数，总和精确等于 W

4. 期望成本计算：
   expectedCost = Σ(weight_i × cost_i) / W

5. 价格计算：
   自动模式：price = expectedCost / (1 - M/100)
   手动模式：M_actual = (W × price - Σ(weight×cost)) / (W × price)
```

### 5. 保底计算

基于概率模型，对每个等级：

```
P = 等级总权重 / 总权重
保底(n) = ceil(ln(1 - n/100) / ln(1 - P))
```

显示 50%、80%、95%、99% 四个置信度的保底抽数。

### 6. Excel 导出

生成 CSV 文件（UTF-8 BOM + Excel 兼容逗号分隔），包含：

| 序号 | 产品ID | 产品名称 | 成本价(¥) | 等级 | 权重 |

区域2 — 等级汇总：
| 等级 | 等级权重 | 概率 | 期望抽数 | 50%保底 | 80%保底 | 95%保底 | 99%保底 |

区域3 — 全局概览：
| 总权重 | 单抽价格 | 期望成本 | 利润率 | 每抽毛利 | 产品总数 |
